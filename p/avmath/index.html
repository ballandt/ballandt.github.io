Avmath math module

Avmath is a Python module for mathematical purposes. It contains functionalities for simple mathematical usage as sine or logarithm functions and submodules for more advanced applied math in the topics of analysis and linear algebra. The module uses a mathematical syntax. The classes and functions are named in a mathematical manner and the class operations enable a mathematical workflow. Its issue is to calculate as accurate as possible and not firstly the aspect of time.

Features

Basic features

Fractions

Avmath provides a datatype for fractions for the purpose of easy calculations. It can be used with other number types because it contains methods for mathematical operations with the built-in operators as +, -, * and /. Fractions with integer values can be reduced.

Functions

The module contains various functions for arithmetic usage as the greatest common divisor or the detection of prime numbers. It also gives access to mathematical functions as the sine or cosine or logarithms of a real number.

Constants

Avmath enables the usage of mathematical constants as π or e or a list of all two digit prime numbers.

Linear algebra

The linear algebra is an important mathematical domain and has many usages in programming. The avmath submodule algebra deals with algebraic objects and their relations.

Tuples

Algebraic tuples can be used to store and manipulate number values or serve as point in geometrical constructions.

Vectors

Vectors are useful objects for calculations in mathematics and natural sciences. The avmath algebra vector class provides a wide span of features for vector operations.

Matrices

Avmath matrices contain many important methods for an easy handling of complicated matrix operations. The algorithms can serve for simple matrix addition or multiplication for the calculation of determinants and solve eigenvalue problems.

Systems of linear equations

SLEs are a part of many mathematical or engineering problems. The library provides easy-to-use features for the declaration and solution of the systems.

Analysis / Calculus

The field of calculus has wide possibilities of usage in many aspects of programming. Avmath helps with easy usable tools for many calculations.

Common functions

Any imaginable function f(x) can be implemented in your program using the avmath analysis function. The function terms can be given as strings to give a maximum of flexibility. The class supports combinations of functions finds roots, extrema and can calculate derivatives and certain integrals.

Polynomial functions

Polynomials are one of the most used function types. Avmath performs addition and multiplication methods finds real and complex roots of polynomials and can calculate extrema. The methods for infinitesimal calculations contain derivative values and functions and certain and incertain integrals.

Version history

Avmath 3 introduced the possibility to accurately calculate objects inlinear algebra using fractions. Many obsolete parts were deleted. Many bugs of version 2 were closed and new features implemented. Avmath 2 ended the former dependency to the math module. All mathematical problems are now solved independently. There have been changes in function names and parameters because of necessary syntax changes and corrections in names. Also, the implementation of functions was done. The Taylor-functions sometimes lack of speed andaccuracy. Many functions and methods had visible and invisible bugs. Avmath 1 was the first GitHub version of avmath. It had the basic functionalities of the later avmath, but especially advanced matrix and function features were missing.

Even though avmath 1 was the first version on GitHub, there existed older versions of avmath - or former: evmath. The last version of evmath can be found on PyPi too. Yet there was no documentation of changes.

The first drafts of avmath are dated on the 22nd of March 2021. Visit CHANGELOG
