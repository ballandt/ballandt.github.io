from avmath import algebra

Tuples

In the linear algebra, tuples are an ordered set of values ak with a = (a0, a1, ..., an) ∈ ℜn. Avmath algebra provides an own Tuple class to handle algebraic operations.

algebra. Tuple

Attributes

Tuple. _value

List that stores the values ak of the Tuple.

Methods

Processing methods

Tuple. __init__(*args)

Initialises the Tuple. Parameters can either be number values or a list. 

from avmath import algebra

# Initialisation via numbers
a = algebra.Tuple(1, 2, 4)

# Initialisation via iterable object
b_list = [4, 3, 4.9]
b = algebra.Tuple(b)

Note: The Tuples a and b will be used in various examples in which this code snippet is used.

Tuple. __iter__()

Returns generator to convert Tuple to another iterable object. Enables the usage of iter function and type conversion to iterable datatypes and the usage in for-loops. 

                
# Usage of iter function
a_iterable = iter(a)

# Usage of list function
a_list = list(a)

# Usage of for loop
for ele in a:
    print(ele)
Tuple. __getitem__(item)

Returns the itemth item of a Tuple. Enables brackets notation: 

a_k = a[1]

Tuple. __repr__()

Method that returns string representation of the Tuple. String representation corresponds with the built-in tuple. This method enables the usage of repr str and print functions.

Tuple. __len__()

Returns the length of a Tuple (equal with its dimension). Enables the usage of the built-in len function.

Tuple. dim()

See Tuple.__len__()

Tuple. append(value)

Returns a Tuple with the appended value. The initial Tuple is not changed.

Tuple. no_fractions()

Converts all Fraction members to floats.

Mathematical methods

Tuple. __eq__(other)

Method for the == operation.

Tuple. __neg__

Returns negative Tuple. All elements are multiplied with -1. Enables the usage of 

negative_a = -a
Tuple. __add__(other)

Method for addition with + operator. Adds two Tuples c = a + b with c = (a0 + b0, a1 + b1, ..., an + bn) and a, b, c ∈ ℜ;n.

Tuple. __sub__(other)

Method for subtraction with - operator. Subtracts two Tuples a - b ≔ a + (-b).

Tuple. __mul__(other)

Method for multiplication with * operator. Calculates scalar multiplication of t · a ≔ (t · a0, t · a1, ..., t · an) with t ∈ ℜ, a ∈ ℜn. Order for operation is not important. 

t = 2
t_a = t * a 
a_t = a * t 
t_a == a_t # True
Tuple. __truediv__(other)

Method for division with / operator. Scalar multiplication with 1 / other. Members ak become Fractions to avoid inaccuracies due to float division.

Static methods

Tuple. dim_check(*args)

Checks if given Tuple arguments have the same amount of dimensions. In this case returns True, else False.

Vectors

The linear algebra does not see any difference between a tuple and a vector. Yet, in geometrical constructions it can be helpful to distinguish between points and sides of a figure. In avmath, the Tuple is generally seen as a point and a Vector as a side. The vector class provides also some additional typical vector features.

algebra. Vector

The Vector class inherites all methods from the Tuple class. If there are no methods to overwrite them, they can be used.

Methods

Processing methods

Vector. __init__(*args [, begin, end])

Initialisation of a Vector object. Takes either number arguments, vectorizes a Tuple or is initialised by giving a point of beginning and end as tuples. In this case, the specific begin and end values are not saved. 

from avmath import algebra

# Initialisation via numbers
u = algebra.Vector(1, 2, 4)

# Initialisation via vectorisation
a = algebra.Tuple(2, 5, 2)
v = algebra.Vector(a)

# Initialisation with begin and end
b = algebra.Tuple(-1, 2, 0)
c = algebra.Tuple(4, 5, 8)
w = algebra.Vector(begin=b, end=c)
Vector. leading_zeros()

Returns the number of the leading zeros of a vector. Important for matrix Gauss-methods. 

u = algebra.Vector(0, 2, 4)
v = algebra.Vector(1, 0, 0)
w = algebra.Vector(0, 0, 0)

leading_zeros = u.leading_zeros() # 1
leading_zeros = v.leading_zeros() # 0
leading_zeros = w.leading_zeros() # 3
Vector. no_fractions()

Returns vector with no Fraction members.

Mathematical methods

Vector. __abs__()

Returns the absolute (pythagorean length) of a vector. The built-in function abs can be used. 

u_abs = abs(u)
Vector. __add__(other)

Method for vector addition. Mathematically identic with tuple addition. + operator can be used. 

sum = u + v
Vector. __sub__(other)

Subtraction method for vectors. Uses a - b = a + (-b). - operator can be used.

Vector. __mul__(other)

  • If other is a number, calculates the scalar multiplication, see Tuple.__mul__
  • If other is another vector, calculates the scalar product a0 · b0 + a1 · b1 + ... + an · bn

Vector. __truediv__(other)

Scalar multiplication with 1/other. Enables usage of Fractions. / sign can be used.

Vector. __pow__(power)

Power method for vectors. Takes only integers [0, ∞) ∈ ℤ as arguments. power=0 returns specific unit vector All other powers return a n times vector. 

e = u ** 0 # unit vector
u_squared = u ** 2 # u * u (a scalar value)
u_cube = u ** 3
# u * u * u (a vector value - scalar (u * u)
# multiplied with a vector gives a vector.
Vector. cross(other)

Returns the cross product or vector product v⃗ × w⃗. Can only be used for vectors v⃗, w⃗ ∈ ℜ3. 

cross_vector = v.cross(w)
Vector. unit()

Returns a vector with same direction, but absolute length 1.

Vector. orthogonal(*args)

Returns a list of the orthogonal projections of the args to the self vector.

Static methods

Vector. spat(u, v, w)

Method for spat product of three vectors (u⃗ × v⃗ ) · w⃗. 

spat = algebra.Vector.spat(u, v, w)
Vector. angle(u, v)

Returns the angle between two vectors in radians. 

angle = algebra.Vector.angle(u, v)

Structures

Structures give the possibility to calculate with geometrical objects giving their edge points.

algebra. Structure

Attributes

Structure. points

A list of the edge points of the structure.

Structure. vectors

A list of the vectors linking the points. The first vector links the first and the second point, the last vector the last point and the first point.

Methods

Processing methods

Structure. __init__(*args)

Initialisation of the structure. Takes Tuple values as arguments representing the edge points.

Structure. flat()

Returns True if an area exists in which all points lie. The method is not very accurate because the algorithm uses square roots and float divisions.

Mathematical methods

Structure. circumference()

Returns the circumference of the structure.

Structure. area([opt])

Returns the area of the structure. opt can be set to 'flat' to throw GeometricalError if the area is not flat. Is not recommended because of the inaccurate flat-algorithm. With no option specified GeometricalWarning is caused if the area is not flat.

Static methods

Structure. triangulate

Calculates the area between three Tuple points.

Matrices

Matrices are tuples of a vector space ℜm×n or so called two dimensional lists. They have a vast importance in many scientific operations. The avmath algebra matrices provide advanced mathematical matrix features. The class inherites from the Tuple class.

algebra. Matrix

Methods

Processing methods

Matrix. __init__(*args)

Initialises the matrix. Input types can be list or algebra.Vector. 

from avmath import algebra

# Initialisation via lists (normal)
A = algebra.Matrix([1, 2, 3], [4, 2, 5], [5, 4, 2])

# Initialisation via lists (better view)
B = algebra.Matrix([1, 2, 3],
                   [4, 2, 5],
                   [5, 4, 2])

# Initialisation via Vector
u = algebra.Vector(1, 2, 3)
C = algebra.Matrix(u)       # Vector becomes column vector
D = algebra.Matrix(list(u)) # Vector becomes row vector
Matrix. __repr__()

Returns a string representation of the matrix. 

A = algebra.Matrix([1, 2, 3],
                   [4, 2, 5],
                   [5, 4, 2])

gives the output:

┌ 1 2 3 ┐
| 4 2 5 |
└ 5 4 2 ┘
Matrix. size([opt])

Returns a list containing the amount of rows and columns of a matrix. With no option specified, returns [nrows, ncolumns]. With opt="xy" the order is inverse (refering to the x-y coordinate system).

Matrix. dim()

Returns the dimension of a matrix. Returns m · n.

Matrix. index(element)

Returns the index of an element in the matrix in [m, n] notation. If there are multiple elements matching the filter, a list with all indices is returned.

Matrix. no_fractions()

Returns a matrix that does not contain Fraction members.

Matrix. column(column_index)

Returns the column vector (return type is algebra Vector) of given index (starting with 0).

Matrix. row(row_index)

Returns the row vector (return type is algebra Vector) of given index (starting with 0).

Matrix. append([row], [, column])

Returns a matrix with appended row and/or column. entered row/column must be an iterable type.

Matrix. remove([row_index], [, column_index])

Returns a matrix without the column and/or row of the given index (starting from 0).

Mathematical methods

Matrix. __round__([n])

Enables usage of built-in round function. All elements are rounded to n digits. If no n specified, all elements are rounded to integers.

Matrix. __neg__()

Returns a negative matrix. All elements are multiplied with -1. Enables the usage of the - operator for negations. 

negative_A = -A
Matrix. __add__(other)

Adds two matrices of the same size C = A + B with Cij = Aij + Bij. Enables + operator. If the matrices do not have the same amount of dimensions, ArgumentError is thrown.

Matrix. __sub__(other)

Reversed matrix addition. Is used by - operator for subtractions.

Matrix. __mul__(other)
  • other is a matrix: returns the matrix product of two matrices A × B with A ∈ ℜm×n and B ∈ ℜn×k. If this condition is not given, raises MatrixError.
  • other is a vector: returns matrix-vector-multiplication. Interpretes vector as a matrix v ∈ ℜn×1 (column vector). If the dimensions do not match, MatrixError is thrown.
  • other is a number: returns scalar multiplied matrix.
Matrix. __truediv__(other)

/ method for matrices. Returns matrix with Fraction members.

Matrix. __pow__(power)

Method for ** operator or pow() function. Returns power times multiplied matrix. Matrix must be quadratic. Power 0 returns identity matrix of the dimension of the operated matrix, power -1 returns inverse matrix.

Matrix. __abs__()

See Matrix.det()

Matrix. transpose()

Returns transposed matrix AT that is the inserted matrix, but the nth row becomes th nth column.

Matrix. det()

Returns the determinant of a quadratic matrix. Calculates with Laplace development.

Matrix. cof()

Returns cofactor matrix B of a quadratic matrix A. Element bij = Det(Aij). Det(Aij) is the determinant of the matrix a with deleted row i and column j.

Matrix. adj()

Returns adjunct of a quadratic matrix. Id equal to the transposed cofactor matrix.

Matrix. inverse()

Returns inverse matrix A-1 of a quadratic matrix A if possible. Can not be processed if the determinant of A is 0. In this case raises ZeroDivisionError.

Matrix. ref()

Returns row echelon form of a matrix.

Matrix. rank()

Returns rank of a matrix.

Matrix. rref()

Returns reduced row echelon form of a matrix.

Matrix. qr()

Returns two matrices that are qr decomposed matrices of the given matrix.

Matrix. eigenvalues([mode=complex])

Returns the complex eigenvalues of a quadratic matrix (a list of complex instances). Mode can be set to mode=float or mode="real" to receive only the real eigenvalues (a list of float intances).

Matrix. eigenvector(eigenvalue)

Returns the eigenvector of a matrix that corresponds with a given eigenvalue.

Static methods

Matrix. create(m, n)

Creates a Matrix of the size m × n containing only 0s.

Matrix. create_identity(n)

Creates an identity matrix with n rows and columns.

Matrix. __leading_zero_sort(arg_list)

Sorts the given matrix value list so that elements are ordered by the numbers of leading zeroes they have.

Systems of linear equations

Systems of linear equations can easily be solved with matrix operations. Avmath algebra provides an own SLE class containing features for their solution. Inherites from Matrix.

algebra. SLE

Attributes

SLE. A

Stores the coefficient matrix.

SLE. b

Stores the result vector.

Methods

Processing methods

SLE. __init__(*args)

Takes arguments as expanded coefficient matrix. 

from avmath import algebra

a = algebra.SLE([a_11, a_12, a_13, b_1],
                [a_21, a_22, a_23, b_2],
                [a_31, a_32, a_33, b_3])

for the system

a11x1 + a12x2 + a13 · x3 = b1

a21x1 + a22x2 + a23 · x3 = b2

a31x1 + a32x2 + a33 · x3 = b3

Mathematical methods

SLE. solve()

Returns a vector with the unknown variables. If the system cannot be solved, throws ZeroDivisionError.

SLE. x(index)

Returns the unknown variable of the given index.
