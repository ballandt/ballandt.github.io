Taylor-series

Introduction

The sine of a number is an essential value for many scientific calculations. It cannot simply be obtained for any real value without the Taylor-series, a theorem by Brook Taylor stating that any \(n\)-times differentiable function \(f\) at \(a\) can be characterised by the polynomial \begin{align} \newcommand\at[2]{\left.#1\right|_{#2}} f(x) = \sum_{n=0}^{\infty} \frac{(x-a)^n}{n!} \cdot \at{\frac{d^ny}{dx^n}}{x=a} \end{align} With this algorithm, many complicated functions can be simplified and calculated numerically.

Algorithm

This example will show the implementation of a certain function, the sine of a number. The Taylor-series will be developed from \(a=0\). Firstly, the second term of the summation formula will be regarded. For any even \(n\) the \(n\) derivative of the sine is either \(y=\sin x\) or \(y=-\sin x\). At the point \(a=0\) both of them are equal to zero which means that every even term of the sum can be neglegated. For every odd \(n\) the term is either \(\cos 0 = 1\) or \(-\cos 0 = -1\), indicating that the odd terms are alternating. The first term can be simplified because \(a\) can be neglegated. This gives the specified Taylor-expansion of the sine of a number \begin{align} \sin x = \frac{x}{1!} - \frac{x^3}{3!} + \frac{x^5}{5!} \pm \cdots \pm \frac{x^n}{n!} = \sum_{k=0}^{\infty} (-1)^{k} \frac{x^{2k+1}}{(2k+1)!} \end{align} Analysing this formula it can be seen that it converges rapidly for small \(x\) values because in this case the factorials in the denominator increase much faster than the exponentials in the numerator. For the sine function this instant can be used because the sine function is periodical in the form that \(\sin x = \sin(x \mod 2\pi)\). Using this period greater values can be avoided.
