Avmath 3.1 documentation - submodule algebra

The algebra submodule contains objects and methods for calculations with linear algebra. It is separated from the root features and must be imported using 

from avmath import algebra

Tuples

In the linear algebra, tuples are an ordered set of values ak with a = (a0, a1, ..., an) ∈ ℜn. Avmath algebra provides an own Tuple class to handle algebraic operations.

algebra. Tuple

Attributes

Tuple. _value

List that stores the values ak of the Tuple.

Methods

Processing methods

Tuple. __init__(*args)

Initialises the Tuple. Parameters can either be number values or a list. 

from avmath import algebra

# Initialisation via numbers
a = algebra.Tuple(1, 2, 4)

# Initialisation via iterable object
b_list = [4, 3, 4.9]
b = algebra.Tuple(b)

Note: The Tuples a and b will be used in various examples in which this code snippet is used.

Tuple. __iter__()

Returns generator to convert Tuple to another iterable object. Enables the usage of iter function and type conversion to iterable datatypes and the usage in for-loops. 

                
# Usage of iter function
a_iterable = iter(a)

# Usage of list function
a_list = list(a)

# Usage of for loop
for ele in a:
    print(ele)
Tuple. __getitem__(item)

Returns the itemth item of a Tuple. Enables brackets notation: 

a_k = a[1]

Tuple. __repr__()

Method that returns string representation of the Tuple. String representation corresponds with the built-in tuple. This method enables the usage of repr str and print functions.

Tuple. __len__()

Returns the length of a Tuple (equal with its dimension). Enables the usage of the built-in len function.

Tuple. dim()

See Tuple.__len__()

Tuple. append(value)

Returns a Tuple with the appended value. The initial Tuple is not changed.

Tuple. no_fractions()

Converts all Fraction members to floats.

Mathematical methods

Tuple. __eq__(other)

Method for the == operation.

Tuple. __neg__

Returns negative Tuple. All elements are multiplied with -1. Enables the usage of 

negative_a = -a
Tuple. __add__(other)

Method for addition with + operator. Adds two Tuples c = a + b with c = (a0 + b0, a1 + b1, ..., an + bn) and a, b, c ∈ ℜ;n.

Tuple. __sub__(other)

Method for subtraction with - operator. Subtracts two Tuples a - b ≔ a + (-b).

Tuple. __mul__(other)

Method for multiplication with * operator. Calculates scalar multiplication of t · a ≔ (t · a0, t · a1, ..., t · an) with t ∈ ℜ, a ∈ ℜn. Order for operation is not important. 

t = 2
t_a = t * a 
a_t = a * t 
t_a == a_t # True
Tuple. __truediv__(other)

Method for division with / operator. Scalar multiplication with 1 / other. Members ak become Fractions to avoid inaccuracies due to float division.

Static methods

Tuple. dim_check(*args)

Checks if given Tuple arguments have the same amount of dimensions. In this case returns True, else False.

Vectors

The linear algebra does not see any difference between a tuple and a vector. Yet, in geometrical constructions it can be helpful to distinguish between points and sides of a figure. In avmath, the Tuple is generally seen as a point and a Vector as a side. The vector class provides also some additional typical vector features.

algebra. Vector

The Vector class inherites all methods from the Tuple class. If there are no methods to overwrite them, they can be used.

Methods

Processing methods

Vector. __init__(*args [, begin, end])

Initialisation of a Vector object. Takes either number arguments, vectorizes a Tuple or is initialised by giving a point of beginning and end as tuples. In this case, the specific begin and end values are not saved. 

from avmath import algebra

# Initialisation via numbers
u = algebra.Vector(1, 2, 4)

# Initialisation via vectorisation
a = algebra.Tuple(2, 5, 2)
v = algebra.Vector(a)

# Initialisation with begin and end
b = algebra.Tuple(-1, 2, 0)
c = algebra.Tuple(4, 5, 8)
w = algebra.Vector(begin=b, end=c)
