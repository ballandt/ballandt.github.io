Numeric integration

Introduction

An integration is a summation of an infinite amount of infinitively small elements. It can be visualised as the area under the graph of the integrated function which is also a prominent field of its application. The analytical integration is often very complicated or not even possible, but following the definition \begin{align} \int_a^b f(x)dx := \sum_{i=0}^\infty f(x_i) \Delta i && \text{for } a \le x_i \le b \end{align} it is possible to numerically approximate the finite integral using finite steps instead of infinitively small ones.
