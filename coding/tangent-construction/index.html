Tangent construction

Introduction

A tangent to a curve at a given point \(P\) is the line that has only one intersection with the curve exactly at the \(P\). If the curve is differentiable, the tangent can be obtained using the derivative of the curve at \(P\).

Algorithm

For this algorithm the curve must be given by a function \(y(x)\) including the point \(P\). The tangent will be seen as a linear function \(y = m\cdot x + n\). \(m\) must be equivalent with the slope of the function \(y(x)\) giving \begin{align} \newcommand\at[2]{\left.#1\right|_{#2}} m \equiv \at{\frac{dy}{dx}}{x=P_x} \end{align} Now reviewing the formula of the linear function \(y\) is given by \(P_y\), \(m\) by the derivative and \(x\) by \(P_x\). The missing \(n\) can be obtained by the reorganisation \begin{align} y &= m\cdot x + n\\ y - m\cdot x &= n \end{align} Now all elements are known to return the tangent.

The function will be given by a python function returning the function value of a given argument. The derivative will be calculated with the finite limit formula \begin{align} \frac{\Delta y}{\Delta x} = \frac{f(x+h)-f(x)}{h} \end{align} A good step is normally at about \(h = 10^{-6}\).

Implementation

def f(x):
    """Function giving the curve"""
    return x**2 - x

def diff(func, x, h=1e-6):
    """Derivative function"""
    return (func(x + h) - func(x)) / h

def tangent(func, P):
    """Function constructing the tangent"""
    x = P[0]
    y = P[1]
    m = diff(func, x)
    n = y - m * x
    return m, n

if __name__ == "__main__":
    print(tangent(f, (2, 2)))

Conclusion

The presented code is a simple preset demonstrating the algorithm of calculating the tangent to a given point. Its accuracy is mainly influenced by the differentiation function. In this example it is built as simple as possible leading to a little accuracy.
