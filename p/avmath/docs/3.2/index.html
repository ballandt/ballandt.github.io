Installation

The avmath module is registered as a PyPi project and can be downloaded via pip command:

pip install avmath

After that, the module can be imported as follows. 

import avmath
from avmath import analysis, algebra

It can also manually been downloaded from GitHub as the current release version or as archive of the active source code using the code button. Then unpack the downloaded files and open command prompt in the unpacked file: 

pip install .

Constants

avmath. e

Euler's number e to 21 correct digits.

avmath. gamma

Euler-Mascheroni constant γ to 21 correct digits.

avmath. phi

Golden ration φ to 21 correct digits.

avmath. pi

Mathematical constant π to 21 correct digits.

avmath. two_digit_primes

Contains a list of all two digit primes (primes > 100). Usefull if checks of primes are needed.

avmath. _TAYLOR_DIFFERENCE

Loops of functions using Taylor-series calculate until the difference of the values is less or equal this value. On default is set to 1e-16. Can be altered if faster loops are needed.

avmath. _MAX_CALCULATION_TIME

Some loops may need a long time for calculation. These loops may not calculate longer as 5 seconds. Can be altered if assigned to a number (time in seconds).

avmath. REAL

List of the name of all types containing real numbers. Used for typehints.

avmath. scope

Dictionary of all mathematical constants and functions needed for analysis.Function.

Functions

Arithmetic functions

avmath. is_prime(x)

Returns True if the integer x is a prime number.

avmath. is_even(x)

Returns True if the integer x is an even number.

avmath. gdc(x, y)

Function returning the greatest common divisor of two integers.

avmath. lcm(x, y)

Function returning the least common multiply of two integers.

avmath. sgn(x)

Function returning the signum function of a real number x. Returns 1 for x > 0, zero for x = 0 and -1 for x < 0.

avmath. fac(x [, opt])

Returns the faculty (x!) of an integer like object. opt can take value "double" for double faculty returning the product of all even numbers ≤ x if x is even or all odd numbers ≤ x if x is odd.

Trigonometric functions

avmath. sin(x)

Returns the sine of x. The function is recommended to use in the domain between -100'000 and 100'000 because of inaccuracies of the modulo operator that increase with the value.

x domain post comma precision
-1e4 to 1e4 12 digits
-1e5 to 1e5 10 digits
-1e6 to 1e6 9-10 digits
avmath. cos(x)

Returns the cosine of x. The function is recommended to use in the domain between -100'000 and 100'000 because of inaccuracies of the modulo operator that increase with the value.

x domain post comma precision
-1e4 to 1e4 12 digits
-1e5 to 1e5 10 digits
-1e6 to 1e6 9-10 digits
avmath. tan(x)

Returns the tangent of x. Inherites inaccuracies from sine and cosine and is therefore recommended to use in the domain between -100'000 and 100'000 and is also inaccurate when aproaching the asymptotes.

x domain post comma precision
-1e4 to 1e4 (normal case) 12 digits
-1e5 to 1e5 (normal case) 10 digits
-1e6 to 1e6 (normal case) 9-10 digits
1e-3 difference to asymptote 10 digits
1e-4 difference to asymptote 7 digits
avmath. arcsin(x)

Function returning the arc sine of a number -1 ≤ x ≤ 1. Calculation time increases when aproaching x → |1|. Function may be stopped by _MAX_CALCULATION_TIME.

Difference of x-value from |1| post comma precision
≥ 1e-1 15 digits
≥ 1e-2 13 digits
≥ 1e-3 4
avmath. arccos(x)

Function returning the arc cosine of a number -1 ≥ x ≥ 1. Due to the usage of arcsin, the accuracy stays the same.

x domain post comma precision
-1e4 to 1e4 12 digits
-1e5 to 1e5 10 digits
-1e6 to 1e6 9-10 digits
avmath. arctan(x)

Function that returns the arc tangent of x. Calculation time increases when approaching x → |1|.

absolute of x domain post comma precision
0 to 0.9 14-15 digits
|1 - x| ≥ 1e-2 13-14 digits
|1 - x| ≥ 1e-3 13 digits
1 6 digits

x domain post comma precision

Function that returns the hyperbolic tangent of x. If x > 20 returns sgn(x) · 1.

Function returning the inverse hyperbolic sine Areasinus hyperbolicus of a number. Domain is restricted to -1e150 ≤ x ≤ 1e150 else causes OverflowError.

Fractions

avmath. Fraction

Attributes

Fraction. a

Numerator of the fraction.

Fraction. b

Denominator of the fraction. Is always made positive. If input was a negative value, the numerator self.a is multipied with -1.

Methods

Processing methods

Fraction. __init__(numerator, denominator)

Initialises Fraction with given numerator and denominator. Arguments can be int float or Fraction. 

import avmath

my_fraction = avmath.Fraction(3, 7)

The Fraction is now initialised and ready to use.

Fraction. __repr__()

Returns a string representation of the fraction. Tries to return reducsd values. Built-in reprmethod can be used and this method is called by the print function. 

import avmath

a = avmath.Fraction(2, 4)
b = avmath.Fraction(6, 3)
print(a)
print(b)

Gives the output 

1/2
2
Fraction. int_args()

Returns True if numerator and denominator are int values. Else returns False.

Mathematical operations

Fraction. __neg__()

Negation of a fraction. Sets numerator a to -a.

Fraction. __eq__(other)

Method for == operation. Can be used with REAL values.

Fraction. __lt__(other)

Method for < (less than) operation. Can be used with REAL values.

Fraction. __gt__(other)

Method for > operation. Can be used with REAL values.

Fraction. __add__(other)

Method for + operation. Can be used with REAL values. Return value is always a Fraction.

Fraction. __sub__(other)

Method for - (subtraction) operation. Can be used with REAL values. Return value is of Fraction type.

Fraction. __mul__(other)

Method for * (multiplication) operation. Can be used with REAL values. Returns Fraction value.

Fraction. __truediv__(other)

Method for / (mathematical division) operation. Can be used with REAL values. Return type is always Fraction.

Fraction. __pow__(power)

Method for ** or built-in pow function (power operation). Can be used with REAL values. Returns a Fraction value. If power has the value -1, numerator and denominator are switched.

Fraction. __rpow__(other)

Power method with Fraction only as exponent. Can be used with REAL values. Returns float.

Fraction. __mod__(other)

Modulo operation used by %. Returns float value.

Fraction. __abs__()

Returns absolute value of the Fraction. Used by built-in abs function. Numerator is made positive.

Fraction. reduce()

Reduces Fraction if both arguments are integers. Else raises ArgumentError.

Type conversion methods

Fraction. __int__()

Returns an integer of the Fraction. Is used by built-in int function. Cuts post comma digits.

Fraction. __float__()

Returns float representation of the Fraction. Is used by built-in float function.

Fraction. __complex__()

Returns complex representation of the Fraction. Is used by built-in complex function. Real part is calculated with __float__ method.

Errors and Warnings

Avmath provides its own errors representing specific issues in the usage.

Errors

avmath. ArgumentError

The argument given to a function or method does not match the filters. Is thrown in case of domain errors or false types.

avmath. DimensionError

The arguments of an operation have a different amount of dimensions and therefore cannot be combined. Especially used by avmath.algebra operations.

avmath.algebra. GeometricalError

A geometrical construction cannot be done due to false arguments.

avmath.algebra. MatrixError

Matrix arguments for an operation is inappropriate. Often handles size issues.

Warnings

avmath.algebra GeometricalWarning

A geometrical instruction is possible, but there may be unwanted conditions. Can be disabled running the method 

GeometricalWarning.disable()
